# Manual-Test

Key Features:

Test Cases: The repository includes a well-defined set of test cases covering various aspects of Flipkart, including user registration, Product search and browse, Add to cart, payment gateways functions. Test cases are categorized by in 4 module and user role for easy navigation and execution.

Expected Results: Every test case clearly defines expected results for each interaction. This helps testers identify bugs, regressions, and inconsistencies in OpenCart's behavior.

Actual Results: Every test case also clearly defines actual results for each interaction. 
This helps tester to check actual results is as expected an expected results. It helps the tester to identify the actual results. R

Test Report: Since no bugs are not incounterd while testing and all cases are executed correctly without any bugs. This include the function, description, expected test case result, actual test case result, pending test cases and remarks.
